# QuickStart: Core UI Features

## How to Test

### Inventory

1.  Navigate to **Inventory**.
2.  Use **Receive Goods** (if PO exists) or **Adjust Stock**.
3.  Verify stock count updates immediately.

### Payments

1.  Navigate to **Finance > Invoices** (or Bills).
2.  Click an Invoice to view details.
3.  Scroll to **Payment History** to see past transactions.
4.  Record a new payment and verify it appears in the list.

### Team

1.  Navigate to **Team Management** (Sidebar).
2.  Click **Invite User**. Enter a test email.
3.  Verify the user appears in the list.
4.  Use **Assign** to add them to a company.
