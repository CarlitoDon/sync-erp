# Implementation Plan: Interactive Getting Started Guide

**Branch**: `17-interactive-onboarding` | **Date**: 2025-12-14 | **Spec**: [spec.md](./spec.md)  
**Input**: Feature specification from `/specs/017-interactive-onboarding/spec.md`

## Summary

Transform the static "Getting Started" checklist on the Dashboard into an interactive onboarding guide. Each step will be clickable (navigates to relevant page), show data-driven completion status, and include a progress indicator. This is a **frontend-only feature** - no new API endpoints required since all data is already available from existing endpoints.

## Technical Context

**Language/Version**: TypeScript 5.x, React 18.x  
**Primary Dependencies**: react-router-dom (navigation), existing dashboardService  
**Storage**: N/A (uses existing API data)  
**Testing**: Browser testing via dev server  
**Target Platform**: Web (Chrome, Firefox, Safari, Edge)  
**Project Type**: web (Vite + React)  
**Performance Goals**: Instant navigation (<100ms), real-time progress updates on refresh  
**Constraints**: Must use existing data endpoints, no new API calls  
**Scale/Scope**: Single component enhancement, 6 onboarding steps

## Proposed Changes

### Component: Dashboard Feature

#### [NEW] OnboardingGuide.tsx

New component to render interactive onboarding checklist:

- Accepts `metrics` prop to determine step completion
- Each step is clickable (Link to target page)
- Shows progress bar with "X of Y completed"
- Supports expand/collapse for step details
- Dismissible after all steps complete

#### [NEW] useOnboardingProgress.ts

Custom hook to calculate onboarding status:

- Takes dashboard metrics as input
- Returns array of OnboardingStep with completion status
- Returns overall progress (completed/total)
- Memoized to prevent unnecessary recalculations

#### [MODIFY] Dashboard.tsx

Replace the current static "Getting Started" section with `<OnboardingGuide />` component.

#### [MODIFY] types.ts

Add types:

- `OnboardingStep`: id, title, description, targetPath, isCompleted
- `OnboardingProgress`: steps[], completedCount, totalCount, isAllComplete
